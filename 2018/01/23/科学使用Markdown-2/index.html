· 前言 ：

MarkdownPad 是 Windows 平台下一款优秀的 Markdown 编辑器，本文简单介绍 Markdown 以及使用一种方法破解 MarkdownPad 使其升级到专业版。该方法仅限于教育用途，我不为滥用该方法导致的商业纠纷负责！

1. 准备：

  • Markdownpad 2：官网下载

  • HxD：官网下载

2.操作：

①：以管理员身份运行[敲黑板] HxD

②：ctrl+o，找到并打开MarkdownPad2.exe

③ ctrl+f，将数据类型选择为“十六进制”，在搜索框内粘贴2C 02 16 2A 02 02 03

④ 找到16，并修改成17

⑤ 重启markdown便可以科学使用了~~
